# Требования: Одностраничный сайт маркетингового агентства

## Статус: ПРОТОТИП (название и цвета будут добавлены позже)

---

## 1. Общее описание

Одностраничный (landing page) сайт для маркетингового агентства.
Стек: **HTML + CSS + Vanilla JS + GSAP** (анимации).
Адаптивный дизайн: desktop, tablet, mobile.

---

## 2. Визуальная концепция

### Тема: "Эффект погружения в глубину"
- **Hero секция** — светлый фон (белый/очень светлый серый)
- Каждая следующая секция **плавно темнеет**
- **Footer / форма заявки** — полностью тёмный фон (почти чёрный)
- Переход реализован через CSS gradient между секциями + GSAP ScrollTrigger для плавности
- Текст автоматически адаптируется: тёмный текст на светлых секциях → светлый текст на тёмных

### Типографика
- Шрифт: пока **Inter** (Google Fonts), можно заменить позже
- Заголовки: bold/semibold, крупные
- Текст: regular, 16-18px базовый размер

### Цвета (временные, будут заменены)
- Акцентный: `#6C63FF` (фиолетовый) — placeholder, заменим на цвет бренда
- Светлый фон: `#FAFAFA` → переход → `#0A0A0A` тёмный
- Текст: `#1A1A1A` (светлые секции) → `#F5F5F5` (тёмные секции)

---

## 3. Интерактивные фичи (из XSpace)

### 3.1 Кастомный курсор
- **Двойной круг**: маленький заполненный inner + большой контурный outer
- Inner следует за мышью мгновенно, outer — с задержкой (lerp эффект)
- При наведении на кликабельные элементы (ссылки, кнопки):
  - Outer увеличивается
  - Возможен mix-blend-mode эффект
- Скрывается на мобильных устройствах (только desktop)

### 3.2 Fullscreen меню (боковое)
- Гамбургер-иконка в header (фиксированная)
- При клике — fullscreen overlay с анимацией (slide-in или fade)
- Внутри: навигационные ссылки (О нас, Услуги, Портфолио, Заявка)
- Плавная прокрутка к секции при клике на ссылку
- Меню закрывается после выбора пункта

## 4. Структура секций (сверху вниз)

### 4.0 Боковая панель (fixed, правый край)
- **Позиция:** фиксирована справа, вертикально по центру экрана (position: fixed, right: 0)
- **Всегда видна** при любом скролле, на всех секциях
- **3 кнопки-иконки, расположены вертикально:**
  1. **Telegram** — иконка Telegram, ссылка на канал (placeholder: t.me/channel)
  2. **Instagram** — иконка Instagram, ссылка на аккаунт (placeholder: instagram.com/account)
  3. **Наверх** — стрелка вверх, плавная прокрутка к началу страницы (scrollTo top)
- **Стиль:**
  - Маленькие круглые или скруглённые кнопки (~40-48px)
  - Полупрозрачный фон с blur или лёгкая тень
  - Hover-эффект: увеличение / подсветка акцентным цветом
  - Цвет иконок адаптируется к фону секции (светлые иконки на тёмных секциях и наоборот)
- **Мобильная версия:** скрывается или перемещается в footer / горизонтальную полосу снизу

### 4.1 Header (sticky с анимацией приклеивания)
- Логотип (слева) — placeholder текст пока
- Гамбургер-кнопка (справа) для fullscreen меню
- **Поведение при скролле (анимация "мультяшные ручки"):**
  1. Изначально header — часть страницы (position: relative), находится вверху Hero
  2. При первом скролле вниз (когда header уходит за viewport) — запускается анимация:
     - Header начинает подниматься вверх вместе со скроллом
     - По бокам header (или сверху по краям) **появляются две маленькие мультяшные ручки** (SVG/CSS-анимация)
     - Ручки "цепляются" пальцами за верхний край экрана (viewport top)
     - Ручки **подтягивают header вверх** — header как будто "карабкается" и прилипает к верху экрана
     - После приклеивания ручки плавно исчезают (fade-out или убираются за край)
  3. После анимации header фиксируется (position: fixed, top: 0) с backdrop-blur фоном
  4. Header **остаётся зафиксированным** при всех последующих скроллах
  5. **Обратная анимация (скролл вверх до Hero):**
     - Когда пользователь возвращается к самому верху страницы (Hero секция полностью видна)
     - Ручки снова появляются — на этот раз "отцепляют" header от верха экрана
     - Ручки как будто **отпускают grip**, разжимают пальцы
     - Header плавно "опускается" обратно на своё исходное место в потоке страницы
     - Ручки исчезают, header возвращается в position: relative
     - Цикл может повторяться: скролл вниз → приклеивание, скролл вверх → отклеивание
- **Детали ручек:**
  - Стиль: мультяшные, минималистичные (простые контурные руки с пальцами)
  - Размер: небольшие, не перекрывают контент header
  - Расположение: по краям header (слева и справа) или две руки по центру сверху
  - Реализация: SVG-элементы + GSAP анимация (grab → pull → release → fade-out)
  - Длительность всей анимации: ~0.8-1.2 секунды
- На светлых секциях — прозрачный/светлый фон; на тёмных — адаптируется

### 4.2 Hero секция
- Большой заголовок с анимацией появления (GSAP SplitText эффект по словам)
- Подзаголовок — краткое описание агентства

## 5. Анимации (GSAP)

| Элемент | Анимация | Триггер |
|---------|----------|---------|
| Header | Ручки цепляются за верх → подтягивают header → приклеивают. При возврате наверх — ручки отцепляют header обратно. Циклическая анимация | Скролл вниз (выход за viewport) / скролл вверх (возврат к Hero) |
| Hero заголовок | Появление по словам/буквам | При загрузке |
| Счётчики "О нас" | Числа считают от 0 до значения | ScrollTrigger |
| Карточки услуг | Fade-in + slide-up поочерёдно | ScrollTrigger |
| Карточки портфолио | Scale-in или fade-in stagger | ScrollTrigger |
| Форма | Fade-in снизу | ScrollTrigger |
| Переход фона | Плавное затемнение между секциями | ScrollTrigger |

## 6. Адаптивность

| Breakpoint | Поведение |
|-----------|-----------|
| Desktop (1200px+) | Полная версия, все анимации, кастомный курсор |
| Tablet (768-1199px) | Сетка 2 колонки, курсор скрыт, touch-friendly |
| Mobile (до 767px) | 1 колонка, гамбургер-меню, упрощённые анимации |
